编译方法

linux下安装gcc

centos   yum -y install gcc

debian   apt-get -y install gcc


gcc -o 编译后名称 编译文件名称

例如 gcc -o JuLi.bin test1.c




名称你们随便

脚本默认名称 JuLi.bin
脚本默认启动名称   ./JuLi.bin
如需更改此变量请搜索  Start_Check_Scripts   修改此变量   但必须以  ./   方式启动    bash命令无法启动
